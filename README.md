# InfiniteCalculator
시스템프로그래밍기초 무한수 계산기 프로젝트

### 빌드
```
make
```

### 사용
```
./main <최대 출력할 소수점 자리수>
<수식>
```

### 구현 기능
* 덧셈
* 뺄셈
* 곱셈
* 나눗셈(사칙연산 모두 계산시 오차 없음)
* 올바른 수식인지 확인
